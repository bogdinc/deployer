Необходимый софт:

1. python 3.7+
2. модули для питона: web3py, openpyxl, solcx
   установка командой - pip install web3 openpyxl py-solc-x

запускать командой python3 deploy.py


Параметры конфига:
"TESTMODE" - true или false - выбор сети - тестовая при true,
"PRIVATE_KEY" - приватный ключ от кошелька, с которого будет рассылка. На балансе должны быть токены и БНБ
"INPUT_FILE" - входящий эксель файл с данными по контрактам
"GAS_LIMIT" - лимит газа для деплоя (можно не менять, только если контракты будут большими, можно увеличить)
"GAS_PRICE" - цена газа в гвеях. 10 для тестовой сети, 5 для main
"NODES" - адреса нод для тестовой и основной сети(менять не нужно)

}
